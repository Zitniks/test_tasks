# Скриншоты работающего приложения

## 1. Swagger UI - Документация API

![Swagger UI](images/Снимок%20экрана%202026-01-21%20в%2022.09.39.png)

Интерактивная документация API доступна по адресу `http://localhost:8000/docs`. Здесь можно:
- Просмотреть все доступные эндпоинты
- Протестировать API прямо в браузере
- Увидеть схемы запросов и ответов

## 2. Создание короткой ссылки через Swagger UI

![Создание короткой ссылки](images/Снимок%20экрана%202026-01-21%20в%2019.18.44.png)

Демонстрация создания короткой ссылки через Swagger UI:
- Эндпоинт: `POST /api/v1/shorten`
- Входные данные: `{"original_url": "https://www.example.com"}`
- Ответ содержит: `short_url`, `original_url`, `short_code`

## 3. Создание короткой ссылки с кастомным кодом

![Кастомный код](images/Снимок%20экрана%202026-01-21%20в%2019.19.10.png)

Пример создания короткой ссылки с человекочитаемым кастомным кодом:
- Входные данные: `{"original_url": "https://www.google.com", "custom_code": "google-link"}`
- Результат: короткая ссылка `http://localhost:8000/google-link`

## 4. Редирект по короткой ссылке

![Редирект](images/Снимок%20экрана%202026-01-21%20в%2019.21.10.png)

Демонстрация работы редиректа:
- Запрос: `GET http://localhost:8000/{short_code}`
- Ответ: HTTP 302 Found с заголовком `Location`, указывающим на оригинальный URL
- Браузер автоматически перенаправляет на оригинальный URL

## Дополнительные скриншоты

### Health Check эндпоинт

```bash
curl http://localhost:8000/api/v1/health
```

**Ответ:**
```json
{
  "status": "ok"
}
```

### Пример работы через cURL

```bash
# Создание короткой ссылки
curl -X POST "http://localhost:8000/api/v1/shorten" \
  -H "Content-Type: application/json" \
  -d '{"original_url": "https://www.example.com"}'
```

**Ответ:**
```json
{
  "short_url": "http://localhost:8000/abc123",
  "original_url": "https://www.example.com",
  "short_code": "abc123"
}
```

### Проверка редиректа

```bash
curl -I http://localhost:8000/abc123
```

**Ответ:**
```
HTTP/1.1 302 Found
Location: https://www.example.com
```

## Тестирование

### Запуск тестов

```bash
pytest
```

### Покрытие кода

```bash
pytest --cov=app --cov-report=html
```

Отчет о покрытии доступен в директории `htmlcov/`.

Для удобного тестирования API создана коллекция Postman:
