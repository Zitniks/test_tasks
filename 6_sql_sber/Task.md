# Тестовое задание Сбербанк: SQL #103

## Условие

Вывести список имён сотрудников, получающих большую заработную плату, чем у непосредственного руководителя.

## Входные данные

- Таблица **departments** (id, name) — отделы.
- Таблица **employees** (id, department_id, chief_id, name, salary) — сотрудники.
  - **chief_id** — идентификатор непосредственного руководителя (ссылка на employees.id).
  - **salary** — заработная плата.

## Требования к результату

- В результирующей таблице одно поле: **name** (имя сотрудника).
- Учитываются только сотрудники, у которых есть руководитель (chief_id IS NOT NULL).
- В выборку попадают только те сотрудники, чья зарплата **строго больше** зарплаты руководителя.
- Порядок: по имени (ORDER BY name).

## Формат ожидаемого результата

| name |
|------|
| Aaron Vaughan |
| April Goodwin |
| ... |

Конкретный список имён зависит от данных в таблице employees.
