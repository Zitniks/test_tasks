# Скриншоты работающего приложения

## 1. Главная страница Hacker News через прокси

![Главная страница](screens/главная%20страница.png)

Прокси отображает главную страницу Hacker News с модифицированным текстом - после каждого слова из 6 букв добавлен символ ™.

**Что видно на скриншоте:**
- Заголовок "Hacker™ News" (слово "Hacker" из 6 букв получило ™)
- Слова "points™", "submit™" и другие слова из 6 букв модифицированы
- Ссылки ведут через прокси (адрес в браузере: http://127.0.0.1:8232)

## Пример модификации текста

**Оригинал:**
```
The visual description of the colliding files, at
http://shattered.io/static/pdf_format.png, is not very helpful
in understanding how they produced the PDFs, so I took apart
the PDFs and worked it out.
```

**Через прокси:**
```
The visual™ description™ of the colliding files, at
http://shattered.io/static/pdf_format.png, is not very helpful
in understanding how they produced the PDFs, so I took apart
the PDFs and worked™ it out.
```

## Проверка работы

### Запуск прокси

```bash
uvicorn cmd.server.main:app --host 127.0.0.1 --port 8232
```

### Открытие в браузере

```
http://127.0.0.1:8232
```

### Проверка модификации

Откройте любую страницу HN через прокси и проверьте, что:
- ✅ Слова из 6 букв имеют символ ™
- ✅ Ссылки ведут через прокси
- ✅ Страница отображается корректно
- ✅ Формы работают
- ✅ Браузер остается на адресе прокси при навигации
