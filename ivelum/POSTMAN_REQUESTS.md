# Запросы для Postman

## Импорт коллекции

Импортируйте файл `postman_collection.json` в Postman или используйте запросы ниже.

## Базовый URL

```
http://127.0.0.1:8232
```

## Запросы

### 1. Главная страница

**GET** `http://127.0.0.1:8232/`

Получить главную страницу Hacker News через прокси.

**Что проверить:**
- Страница отображается корректно
- Слова из 6 букв имеют символ ™
- Ссылки ведут через прокси

---

### 2. Страница статьи

**GET** `http://127.0.0.1:8232/item?id=13713480`

Получить страницу конкретной статьи.

**Пример из задания:**
- Оригинал: "The visual description of the colliding files"
- Через прокси: "The visual™ description™ of the colliding files"

**Что проверить:**
- Текст модифицирован правильно
- Все слова из 6 букв имеют ™
- Ссылки работают через прокси

---

### 3. Страница комментариев

**GET** `http://127.0.0.1:8232/item?id={item_id}`

Замените `{item_id}` на реальный ID статьи.

**Что проверить:**
- Комментарии отображаются
- Текст в комментариях модифицирован
- Ссылки на профили пользователей ведут через прокси

---

### 4. Страница пользователя

**GET** `http://127.0.0.1:8232/user?id={username}`

Замените `{username}` на имя пользователя (например, `pg`).

**Что проверить:**
- Профиль пользователя отображается
- Текст модифицирован
- Ссылки на статьи ведут через прокси

---

### 5. Поиск

**GET** `http://127.0.0.1:8232/search?q={query}`

Замените `{query}` на поисковый запрос.

**Пример:**
```
GET http://127.0.0.1:8232/search?q=python
```

---

### 6. Новые статьи

**GET** `http://127.0.0.1:8232/newest`

Получить страницу с новыми статьями.

---

### 7. Популярные статьи

**GET** `http://127.0.0.1:8232/front`

Получить популярные статьи.

---

## Примеры cURL

### Главная страница
```bash
curl http://127.0.0.1:8232/
```

### Страница статьи
```bash
curl http://127.0.0.1:8232/item?id=13713480
```

### Сохранение HTML для проверки
```bash
curl http://127.0.0.1:8232/item?id=13713480 > page.html
```

Затем откройте `page.html` в браузере и проверьте:
- Наличие символа ™ после слов из 6 букв
- Работу ссылок через прокси

## Проверка модификации текста

После получения ответа проверьте в тексте:

1. **Слова из 6 букв должны иметь ™:**
   - `visual` → `visual™`
   - `single` → `single™`
   - `format` → `format™`

2. **Слова не из 6 букв остаются без изменений:**
   - `The` → `The` (3 буквы)
   - `files` → `files` (5 букв)
   - `commands` → `commands` (8 букв)

3. **Ссылки должны вести через прокси:**
   - `/item?id=123` → `http://127.0.0.1:8232/item?id=123`
   - `https://news.ycombinator.com/item?id=123` → `http://127.0.0.1:8232/item?id=123`

## Переменные в Postman

Создайте переменные в коллекции:
- `base_url` = `http://127.0.0.1:8232`
- `item_id` = `13713480` (пример ID статьи)
- `username` = `pg` (пример имени пользователя)
- `search_query` = `python` (пример поискового запроса)

Используйте в запросах: `{{base_url}}/item?id={{item_id}}`

## Примечания

- Прокси работает только для страниц Hacker News
- Внешние ссылки (не на HN) не модифицируются
- Скрипты и стили не изменяются
- Только текстовый контент модифицируется
