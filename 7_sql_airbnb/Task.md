# Тестовое задание Airbnb: SQL #74

## Условие

Выведите идентификатор и признак наличия интернета в помещении. Если интернет в сдаваемом жилье присутствует, то выведите «YES», иначе «NO».

## Входные данные

- Таблица **rooms** (id, home_type, address, has_tv, **has_internet**, has_kitchen, has_air_con, price, owner_id, latitude, longitude) — жильё на сервисе бронирования.
  - **has_internet** — булево поле (true/false или 1/0).

## Требования к результату

- Поля в результирующей таблице: **id**, **has_internet**.
- Используйте конструкцию **AS has_internet** для вывода признака наличия интернета.
- Значение: «YES», если интернет есть; «NO», если нет.
- Порядок: по id (ORDER BY id).

## Формат ожидаемого результата

| id | has_internet |
|----|--------------|
| 1  | YES          |
| 2  | YES          |
| 3  | NO           |
| …  | …            |

Конкретные значения зависят от данных в таблице rooms.
